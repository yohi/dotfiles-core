# Dotfiles Common Rules

## 1. 基本コマンド
- `make setup`: プロジェクトの初期セットアップ（依存関係のインストール、設定の適用）を行います。新しい環境やコンポーネントを追加した後は必ず実行してください。
- `make`: 引数なしで実行すると、利用可能なターゲットの一覧と説明を表示します。

## 2. エージェントへの指示
- このプロジェクトの操作は `Makefile` を通じて行ってください。
- 個別のリポジトリ固有のルールは `AGENTS.md` に記述されていますが、この `DOTFILES_COMMON_RULES.md` が優先されます。
